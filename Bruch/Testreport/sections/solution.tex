\subsection{Klasse Bruch}
Die Klasse Bruch soll Bruchteile darstellen k√∂nnen. Sie implementiert fast alle von Python bereitgestellten magischen Methoden, dadurch kann man sehr angenehm mit dieser Klasse arbeiten. 

\subsection{Beispiele} 
\begin{lstlisting}[language=Python]
print(Bruch(5,4))
\end{lstlisting}
Output: \verb|(5/4)|
\\\\


\begin{lstlisting}[language=Python]
print(float(Bruch(5,4)))
\end{lstlisting}
Output: \verb|1,25|
\\\\


\begin{lstlisting}[language=Python]
print(Bruch(3,2) + 1)
\end{lstlisting}
Output: \verb|(5/4)|
